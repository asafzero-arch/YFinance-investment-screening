# 株式スクリーニング

東証上場銘柄の割安株を検索するスキル。

## このスキルを使うタイミング
以下のような発言があったときに自動的にこのスキルを使う：
- 「割安株を探して」
- 「高配当の銘柄を教えて」
- 「東証で成長株を探したい」
- 「スクリーニングして」

## 実行するコマンド
以下のコマンドをBashで実行する：
```bash
cd C:\Users\hidmi\OneDrive\Documents\YFinance\investment-screening
python scripts/screening.py --preset {preset} --limit {limit} --market {market} --max-scan {max_scan}
```

## 引数の解釈ルール

**preset（スクリーニングの種類）**
- 「割安」「バリュー」「安い」→ `value`
- 「高配当」「配当」「インカム」→ `high-dividend`
- 「成長」「グロース」→ `growth`
- 指定なし → `value`

**market（対象市場）**
- 「プライム」→ `prime`
- 「スタンダード」→ `standard`
- 「グロース」→ `growth`
- 「全市場」「全部」→ `all`
- 指定なし → `prime`

**limit（表示件数）**
- 「10件」「10銘柄」→ `10`
- 指定なし → `10`

**max_scan（スキャン件数）**
- 「全部スキャン」→ `1600`
- 指定なし → `100`

## 結果の解説
コマンド実行後、結果の表を見て以下を補足説明する：
- スコア上位銘柄の注目ポイント
- 業種の偏りがあれば指摘
- 割安の理由として考えられること

## 注意事項
- このツールの結果は投資助言ではない
- データはyfinance経由のため若干の遅延がある
- スキャン件数が多いほど時間がかかる（100件で約1分）
